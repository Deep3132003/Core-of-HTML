1. HTML (Hypertext Markup Language)
Used for creating the basic structure and content of web pages.
2. CSS (Cascading Style Sheets)
Responsible for styling and layout, controlling the visual presentation of HTML elements.
3. JavaScript (JS)
Enables interactivity and dynamic behavior on web pages, manipulating DOM elements and responding to events.
4. Node.js
JavaScript runtime environment that allows server-side execution of JavaScript code.
5. Express.js
Web application framework for Node.js, simplifying the creation of APIs and handling of HTTP requests.
6. React
JavaScript library for building user interfaces, providing a component-based architecture.
7. Angular
Comprehensive front-end framework for building dynamic and complex web applications.
8. Vue.js
Progressive JavaScript framework for building interactive user interfaces.
9. MongoDB
NoSQL database that provides a flexible and scalable solution for storing and retrieving data.
10. MySQL
Relational database management system used for structured data storage and retrieval.
11. PostgreSQL
Powerful open-source relational database known for its extensibility and reliability.
12. GraphQL
Query language and runtime for APIs, providing a more efficient and powerful alternative to REST.
13. RESTful APIs
Architectural style for designing networked applications, typically using HTTP methods for communication.
14. Git
Version control system for tracking changes in code and collaborating with other developers.
15. Docker
Containerization platform for packaging applications and their dependencies into isolated units.
16. Heroku
Cloud platform that simplifies the deployment and scaling of web applications.
