HTML tags with description

1. html: The root element that wraps around all content on the page

2. head: Contains meta-information about the document, such as title, character set, and linked resources.

3. title: Sets the title of the web page, displayed on the browser's title bar or tab.

4. meta: Provides meta-information about the document, like character encoding, viewport settings, etc.

5. link: Used to link external resources like stylesheets to the HTML document.

6. body: Contains the content of the web page, including text, images, links, and other elements.

7. (h1 - h6): Heading elements, from highest importance (h1) to lowest (h6).

8. p: Represents a paragraph of text.

9. a: Defines a hyperlink, allowing navigation to another web page or resource.

10. img: Embeds an image into the page.

11. ul: Creates an unordered list.

12. span: A generic inline container often used for styling a specific portion of text.

13. label: Provides a label for an input element.

14. select: Creates a drop-down list.

15. div: A generic container that groups and styles elements.

16. thead: Groups the header content in a table.

17. td: Represents a table data cell.

18. th: Represents a table header cell.

19. tr: Represents a table row.

20. table: Defines a table.

21. iframe: Embeds a separate document into the current document.
